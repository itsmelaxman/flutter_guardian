# guardian_dependencies

Dependency risk analyzer for Flutter Guardian.

Parses `pubspec.yaml` and flags local reproducibility risks such as loose constraints, prerelease constraints, git sources, and path sources.

See the root [README](https://github.com/itsmelaxman/flutter_guardian/blob/main/README.md) and [User Guide](https://github.com/itsmelaxman/flutter_guardian/blob/main/doc/USER_GUIDE.md).
