# guardian_security

AST-based security analyzer for Flutter Guardian.

Detects committed `.env` files, `flutter_dotenv` imports, runtime `.env` access, `print`, `debugPrint`, JWT-like literals, secret-looking literals, and high-entropy string literals.

See the root [README](https://github.com/itsmelaxman/flutter_guardian/blob/main/README.md) and [User Guide](https://github.com/itsmelaxman/flutter_guardian/blob/main/doc/USER_GUIDE.md).
