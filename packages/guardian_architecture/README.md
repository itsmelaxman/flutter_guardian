# guardian_architecture

Architecture boundary analyzer for Flutter Guardian.

Enforces feature isolation under `lib/features/<feature>` and layer direction rules for `data`, `domain`, and `presentation`.

See the root [README](https://github.com/itsmelaxman/flutter_guardian/blob/main/README.md) and [User Guide](https://github.com/itsmelaxman/flutter_guardian/blob/main/doc/USER_GUIDE.md).
